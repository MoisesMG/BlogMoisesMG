¿Y para que voy a usar JavaScript? ¿Para que usarlo en paginas web?

Bien, este es el el eco mas resonante, con JavaScript haces que tu sitio web sea interactivo, en otras palabras puedas añadir efectos super factasticos como galerias de imagenes dinamicas, botones que cambian de color y tamaño al pasar el click sobre ellos, menus principales con efectos super fantatiscos, añadir y agregar contenido en tiempo real como nuevas articulos que escribas, comentario de otras personas, la lista es inmensa. Y para facilitarte esto existe las API's, una API son funciones codificas en JavaScript por otras programadores que nos ayudan a realizar todas los efectos fantasticos que te menciones.
